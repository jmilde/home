?%

(?% expr & {:as dict})

takes a expr and interleaving vars and values, (partially) evaluates the expr according to the binding.

similar to let, this can be seen as performing a lambda-abstraction immediately followed by a beta reduction (a left-left lambda).

(?% phi a x b y)     (let [a x b y] phi)
(? (% phi a b) x y)  ((fn [a b] phi) x y)

Expr

protocol

an expression satisfies the following protocol.

  • fv returns the set of free variables
  • ev evaluates the expression given dict, a map of variable bindings

expression types and evaluation types.

  • V variable => Expr
  • A application => Expr
  • F function => F
  • _ value => _
